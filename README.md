# PLEASE DO NOT USE WITH DEBRID ADDONS

> [!CAUTION]

## Several media players using the trailer addon format ping *****EVERY***** source provided this will spam your debrid account with requests and cause you to hit your limit very quickly and or be blocked. If you want to use this addon with debrid services, please use it with a media player that does not ping every source provided.

## Trailer Addon

Get the `/manifest.json` url for any ***NON DEBRID*** addon you want streams from
Use this site to base64 encode it url safe **make sure to uncheck "No padding"**.
<https://www.base64.sh/base64url>
Use the url `https://example.com/{base64_addon}/manifest.json`
